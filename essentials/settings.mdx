### Runtime Support

We support multiple modern runtimes so you can run various types of applications without compatibility issues.

- Node.js (LTS & Latest)
- Go (Golang)
- Bun Runtime
- GraphQL Server Environment

All runtimes are optimized for performance and configured for server usage.

### Node.js

We provide full support for Node.js applications including:

- npm / yarn / pnpm support
- ES Modules & CommonJS
- WebSocket support
- Express / Fastify / NestJS compatible
- Long-running bot & API support

Suitable for:
- Telegram bots
- Discord bots
- REST API
- Web apps
- Automation scripts

### Go (Golang)

Our servers support compiled Go applications with high performance.

Features:

- Native binary execution
- Low memory usage
- High concurrency support
- Suitable for API, backend, and microservices

### Bun Runtime

We support Bun runtime for modern JavaScript execution.

Features:

- Faster than Node.js for many workloads
- Built-in bundler
- Built-in package manager
- Native TypeScript support

Recommended for:

- High performance API
- Modern JS apps
- Lightweight services

### GraphQL Environment

We support GraphQL servers with Node.js / Bun runtime.

Compatible with:

- Apollo Server
- Yoga GraphQL
- Mercurius
- Express GraphQL

Suitable for:

- API Gateway
- Backend services
- Modern web apps

### Panel Management

We use **Pterodactyl Panel** for server management.

Features:

- Web-based control panel
- Start / Stop / Restart server
- Console access
- File manager
- Resource monitoring
- Automatic restart
- Secure container isolation

Panel provides easy control for beginners and advanced users.
