## Membuat Database

Ikuti langkah-langkah berikut untuk membuat database baru:

### 1. Klik Tombol New Database
Klik tombol **New Database** pada dashboard.

<Frame>
  ![New Database Button](https://id-cdn.aisbirnusantara.com/documentation/Screenshot_20260328-054535.png)
</Frame>

---

### 2. Masukkan Nama Database
Isi nama database sesuai kebutuhan, lalu lanjutkan proses pembuatan.

![Input Nama Database](https://id-cdn.aisbirnusantara.com/documentation/Screenshot_20260326-072909.png)

---

### 3. Lihat Detail Database
Setelah database berhasil dibuat, Anda akan melihat detail database seperti:
- Nama database
- Username
- Host
- Link akses phpMyAdmin

<Frame>
  ![Detail Database](https://id-cdn.aisbirnusantara.com/documentation/Screenshot_20260326-072909.png)
</Frame>

---

### 4. Akses phpMyAdmin
Gunakan URL phpMyAdmin yang tersedia pada detail database untuk membuka dan mengelola database Anda,.
(Phpmyadmin tersedia pada opsi host cukup hapus :3306 pada domain, misal phpmyadmin.aisbirnusantara.com:3306 -> phpmyadmin.aisbirnusantara.com)

<Frame>
  ![Akses phpMyAdmin](https://id-cdn.aisbirnusantara.com/documentation/Screenshot_20260326-073131.png)
</Frame>
