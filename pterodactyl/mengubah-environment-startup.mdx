### 1. Pilih Opsi Change Egg di Server Subscription

Masuk ke dashboard, lalu buka tab **Subscription** pada server kamu.  
Kemudian pilih opsi **Change Egg**.

![Change Egg](https://id-cdn.aisbirnusantara.com/documentation/Screenshot%202026-03-28%20064329.png)

---

### 2. Pilih Environment

Pilih environment (egg) yang ingin kamu gunakan sesuai kebutuhan aplikasi atau game server kamu.

![Pilih Environment](https://id-cdn.aisbirnusantara.com/documentation/Screenshot%202026-03-28%20064404.png)

---

### ⚠️ Warning

> Mengubah **startup / environment** akan menyebabkan:
>
> - Semua data server **terhapus**
> - File, database, dan konfigurasi lama **tidak dapat dikembalikan**
> - Server akan di-reset ke kondisi awal sesuai environment baru
>
> **Pastikan kamu sudah melakukan backup data penting sebelum melanjutkan.**

---

## Selesai 🎉

Server kamu sekarang menggunakan environment baru sesuai pilihan.
