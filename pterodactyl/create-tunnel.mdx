---

### 1. Pilih Server di Tab Subscription

Masuk ke dashboard, lalu buka tab **Subscription** dan pilih server yang ingin digunakan.

![Pilih Server](https://id-cdn.aisbirnusantara.com/documentation/Screenshot_20260328-060534.png)

---

### 2. Masuk ke Tab Port Forward

Setelah memilih server, buka tab **Port Forward**.

![Tab Port Forward](https://id-cdn.aisbirnusantara.com/documentation/Screenshot_20260328-061112.png)

---

### 3. Pilih Server

Pilih kembali server yang akan digunakan untuk membuat tunnel.

![Pilih Server](https://id-cdn.aisbirnusantara.com/documentation/Screenshot_20260328-061128.png)

---

### 4. Isi Detail Port

Isi detail port sesuai kebutuhan:

- **Port Local**: Port dari aplikasi/server kamu (contoh: 3000, 8080)
- **Port Remote**: Port yang akan diakses dari luar
- **Protocol**: Pilih sesuai (HTTP / TCP)

Setelah itu, simpan konfigurasi.

![Isi Port](https://id-cdn.aisbirnusantara.com/documentation/Screenshot_20260328-061309.png)

---

### 5. Buka URL Tunnel

Setelah konfigurasi berhasil dibuat, kamu akan mendapatkan URL tunnel.

Buka URL tersebut di browser untuk mengakses aplikasi kamu dari internet.

---

## Selesai 🎉

Sekarang tunnel kamu sudah aktif dan bisa digunakan untuk mengakses server secara publik.
