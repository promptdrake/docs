## Server Status

This page explains the meaning of server status shown in the panel.

---

## Installing

Status **Installing** means the server environment is currently being prepared by the system.

This status appears when:

- Creating new server
- Reinstalling server
- Updating runtime
- Changing configuration
- Allocating resources

During this status:

- Server cannot be started
- Files may not be accessible
- Console may not work
- Installation scripts are running

### What you should do

Wait until the installation finishes.

Normal install time:

- Small environment: 5–30 seconds
- Large environment: up to 2 minutes

Do not press reinstall / restart repeatedly while Installing.

### If Installing takes too long

If Installing takes more than 5 minutes:

Contact support and provide:

- Username
- Server name
- Server ID
- Action you did before Installing

---

## Suspended

Status **Suspended** means the server has been disabled temporarily.

This can happen because of:

- Unpaid invoice
- Expired service
- Abuse report
- Resource violation
- Manual suspension by admin

During Suspended status:

- Server cannot start
- Console disabled
- Network blocked
- File access may be locked

### How to fix Suspended

#### Pay invoice

Go to billing page and complete payment.

#### Renew service

If service expired, renew your plan.

#### Contact support

If you think this is an error, contact support and provide:

- Username
- Server name
- Order ID
- Problem description

### Important

Do not send multiple tickets for the same problem.

Support will respond as soon as possible.
