## Status Server

Halaman ini menjelaskan arti dari status server yang ditampilkan di panel.

---

## Installing

Status **Installing** berarti lingkungan server sedang dipersiapkan oleh sistem.

Status ini muncul ketika:

- Membuat server baru
- Menginstal ulang server
- Memperbarui runtime
- Mengubah konfigurasi
- Mengalokasikan resource

Selama status ini:

- Server tidak dapat dijalankan
- File mungkin tidak bisa diakses
- Console mungkin tidak berfungsi
- Script instalasi sedang berjalan

### Apa yang harus dilakukan

Tunggu sampai proses instalasi selesai.

Waktu instalasi normal:

- Environment kecil: 5–30 detik
- Environment besar: hingga 2 menit

Jangan menekan reinstall / restart berulang kali saat status Installing.

### Jika Installing terlalu lama

Jika status Installing lebih dari 5 menit:

Hubungi support dan sertakan:

- Username
- Nama server
- Server ID
- Aktivitas terakhir sebelum Installing

---

## Suspended

Status **Suspended** berarti server dinonaktifkan sementara.

Hal ini bisa terjadi karena:

- Invoice belum dibayar
- Layanan telah expired
- Laporan penyalahgunaan
- Pelanggaran penggunaan resource
- Suspend manual oleh admin

Selama status Suspended:

- Server tidak dapat dijalankan
- Console dinonaktifkan
- Jaringan diblokir
- Akses file mungkin dikunci

### Cara memperbaiki Suspended

#### Bayar invoice

Masuk ke halaman billing lalu selesaikan pembayaran.

#### Perpanjang layanan

Jika layanan expired, lakukan perpanjangan paket.

#### Hubungi support

Jika Anda merasa ini kesalahan, hubungi support dengan menyertakan:

- Username
- Nama server
- Order ID
- Deskripsi masalah

### Penting

Jangan mengirim banyak tiket untuk masalah yang sama.

Support akan merespon secepat mungkin.

---

## Catatan

Jika status server Anda tidak ada di daftar ini, silakan hubungi support.
