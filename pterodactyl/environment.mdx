### Dukungan Runtime

Kami mendukung berbagai runtime modern sehingga Anda dapat menjalankan berbagai jenis aplikasi tanpa masalah kompatibilitas.

- Node.js (LTS & Versi Terbaru)
- Go (Golang)
- Bun Runtime
- Environment Server GraphQL

Semua runtime telah dioptimalkan untuk performa tinggi dan dikonfigurasi khusus untuk penggunaan server.

### Node.js

Kami menyediakan dukungan penuh untuk aplikasi Node.js, termasuk:

- Dukungan npm / yarn / pnpm
- ES Modules & CommonJS
- Dukungan WebSocket
- Kompatibel dengan Express / Fastify / NestJS
- Mendukung bot dan API yang berjalan terus-menerus

Cocok untuk:
- Bot Telegram
- Bot Discord
- REST API
- Web App
- Script otomatisasi

### Go (Golang)

Server kami mendukung aplikasi Go yang dikompilasi dengan performa tinggi.

Fitur:

- Eksekusi binary native
- Penggunaan memori rendah
- Dukungan concurrency tinggi
- Cocok untuk API, backend, dan microservices

### Bun Runtime

Kami mendukung Bun runtime untuk eksekusi JavaScript modern.

Fitur:

- Lebih cepat dari Node.js pada banyak workload
- Bundler bawaan
- Package manager bawaan
- Dukungan TypeScript native

Direkomendasikan untuk:

- API berperforma tinggi
- Aplikasi JavaScript modern
- Service ringan

### Environment GraphQL

Kami mendukung server GraphQL menggunakan runtime Node.js atau Bun.

Kompatibel dengan:

- Apollo Server
- Yoga GraphQL
- Mercurius
- Express GraphQL

Cocok untuk:

- API Gateway
- Backend service
- Aplikasi web modern

### Panel Management

Kami menggunakan **Pterodactyl Panel** untuk manajemen server.

Fitur:

- Control panel berbasis web
- Start / Stop / Restart server
- Akses console
- File manager
- Monitoring resource
- Restart otomatis
- Isolasi container yang aman

Panel ini mudah digunakan baik untuk pemula maupun pengguna lanjutan.

### Infrastruktur

- Region: Indonesia & Eropa
- Storage SSD / NVMe
- Jaringan berkecepatan tinggi
- Proteksi DDoS
- Lingkungan berbasis Linux
- Isolasi container
